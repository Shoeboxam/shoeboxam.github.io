class opendp.v1.mod.Measurement[source]¶

Bases: opendp.v1.mod.LP_AnyMeasurement

A differentially private unit of computation. A measurement contains a function and a privacy relation. The function releases a differentially-private release. The privacy relation maps from an input metric space to an output measure space.

check(d_in, d_out, *, debug=False) → bool[source]¶

Check if the measurement satisfies the privacy relation at d_in, d_out.

Parameters

  • d_in – Distance in terms of the input metric space.

  • d_out – Distance in terms of the output measure space.

  • debug – Enable to raise Exceptions to help identify why the privacy relation failed.

Returns

If True, a release is differentially private at d_in, d_out.

Return type

bool

class opendp.v1.mod.Transformation[source]¶

Bases: opendp.v1.mod.LP_AnyTransformation

A non-differentially private unit of computation. A transformation contains a function and a stability relation. The function maps from an input domain to an output domain. The stability relation maps from an input metric space to an output metric space.

check(d_in, d_out, *, debug=False)[source]¶

Check if the transformation satisfies the stability relation at d_in, d_out.

Parameters

  • d_in – Distance in terms of the input metric space.

  • d_out – Distance in terms of the output metric space.

  • debug – Enable to raise Exceptions to help identify why the stability relation failed.

Returns

True if the relation passes. False if the relation failed.
