Expand description

Fallible multiplication with specified rounding.

Throws an error if the ideal output is not finite or representable.

Required Methods

source

fn inf_mul(&self, v: &Self) -> Fallible<Self>

Proof Definition

For any two values self and v of type Self, self.inf_mul(v) either returns Ok(out), where $out \ge self \cdot v$, or Err(e).

source

fn neg_inf_mul(&self, v: &Self) -> Fallible<Self>

Proof Definition

For any two values self and v of type Self, self.neg_inf_mul(v) either returns Ok(out), where $out \le self \cdot v$, or Err(e).
